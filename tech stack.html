Tech Stack

A tech stack,or technology stack,is the collection of technologies used to build and run a website,application,or project.

LAMP
An industry-standard stack that's cost-effective,flexible and performs well.it's made up of Linux, Apache, MySQL and PHP and is often used to build dynamic websities and web applications
MEAN
A stack that uses open-source technology and Javascript to create websites and interactive applications.
it's made up of MongoDB, Express.js, Angular.js, and Node.js.
Ruby on Rails
A full-stack framework that uses Ruby to simplify the development process.
it's used by large sites like Basecamp, Shopify, Dribble, and GitHub
ASP.NET
A stack that uses Microsoft's.NET core to create lightweight applications for Windows and Linux. it uses many open-source technologies and offers a lot of flexibility to developers
MERN
A full stack Javascript solution that combines MongoDB,Express,React and Node.js
Python-Django Stack
A stack that uses Python and Django to create full-stack applications. it can be used to create blog platforms.
